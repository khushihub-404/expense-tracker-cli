# Expense Tracker CLI

A Python CLI based Expense Tracker application using SQLite, Pandas, and Matplotlib.

## Features

- Add income and expense transactions
- View all transactions
- Monthly summary report
- Export reports to Excel and CSV
- Generate expense charts

## Technologies Used

- Python
- SQLite
- Pandas
- Matplotlib

## How to Run

pip install pandas matplotlib openpyxl

python database.py

python main.py
